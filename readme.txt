download the files

make sure to have the python's latest version installed from the browser

open the folder with app.py in vscode

open terminal

run the following commands:

        pip install virtualenv

        virtualenv env

 [if no env folder created then in the powershell administrator window run: Set-ExecutionPolicy unrestricted and then click A for "yes to all" option  ]
        
        .\env\Scripts\activate.ps1

        pip install flask
        pip install numpy
        pip install tensorflow
        pip install flask-cors

run the web app and flask api: python .\app.py


Now you can run the web extension, to install the web extension:

In chrome browser, open manage extensions and turn on the developer mode

find the load unpacked button and browse the extension folder and select it

the web extension should show up among all the extensions
